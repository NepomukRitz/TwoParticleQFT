# Overview of symbols and notations used throughout these pages

The tables on this page provide an overview of the main symbols and notations used throughout this documentation. We also mention differences to other common conventions in the literature.

## Basic objects

| Object | Symbol used here | Other commonly used symbols |
|:-------|------------------|----------------------|
| Bare propagator | $G_0$ | $G^{0}$ |
| Dressed propagator | $G$ | $g$ (local) |
| Bubble (i.e. propgator pair) | $\chi_0$ | $\chi^{0}$, $\Pi$, $(GG)$ |
| Self-energy | $\Sigma$ | - |
| Bare four-point vertex | $F_0$ | $U$, $V$, $\Gamma_{0}$, $\Gamma^{0}$, $f$ (for dual fermions) |
| Full four-point vertex | $F$ | $\Gamma$, $\Gamma^{(4)}$, $\Gamma^*$, $f$ (local), $V$, $L^X$ |
| Four-point Green's function | $G^{(4)}$ | $g^{(4)}$ (local), $G_2$ |

## Parquet theory

| Object | Symbol used here | Other commonly used symbols |
|:-------|------------------|----------------------|
| two-particle channels | $\overline{ph}$, $pp$, $ph$ | ($a$, $p$, $t$), ($x$, $p$, $d$), (ph,cr; pp; ph,d), ($u$,$s$,$t$) |
| two-particle reducible vertex in channel $r$ | $\Phi^r$ | $\gamma_r$, $\phi_r$, $\Phi_r$ |
| two-particle irreducible vertex in channel $r$ | $\Gamma^r$ | $I_r$, $\Gamma_r$ |
| fully irreducible vertex | $\Lambda$ | $R$, $\tilde{\Lambda}$, $\mathcal{I}$, $\Lambda_{2\mathrm{PI}}$ |

## Single boson exchange (SBE) theory

| Object | Symbol used here | Other commonly used symbols |
|:-------|------------------|----------------------|
| $U$-reducible vertex in channel $r$ | $\Delta^r$ | $\nabla_r$, ($\mathcal{M}$, $\mathcal{C}$, $\mathcal{S}$) |
| $U$-irreducible vertex in channel $r$ | $T^r$ | $\phi^r$, $\mathcal{I}^r$ |
| fully $U$-irreducible vertex | $\Lambda^{U}$ | $\phi^{U_\mathrm{irr}}$, $\phi^\mathrm{firr}$, $\Lambda^{\mathrm{Uirr}}$, $\Lambda_{U \mathrm{irr}}$, $\mathcal{I}_{U \mathrm{irr}}$, $\Phi^\mathrm{Uirr}$ |
| screened interaction in channel $r$ | $W^r$ | $w_r$, $\eta_r$, $W_r$, $D^r$ |
| Hedin vertices in channel $r$ | ($\gamma^r$, $\overline{\gamma}^r$) | ($\lambda_r$, $\bar{\lambda}_r$), ($\Lambda_r$, $\bar{\Lambda}_r$), ($h^r$, $\bar{h}^r$) |
| polarization in channel $r$ | $P^r$ | $\Pi_r$, $\pi_r$, $P_r$ |

| Object | Symbol used here | Other commonly used symbols |

| fermionic frequency | $\nu$ | $\omega$ |
| bosonic momentum | $\mathbf{q}$ | $\mathbf{Q}$ |
| fermionic momentum | $\mathbf{k}$ | - |
